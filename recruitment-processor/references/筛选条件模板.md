## 格式说明
筛选条件采用键值对格式，每行一个条件项。条件之间为"与"关系，即需要同时满足所有条件。

## 条件类型

### 1. 薪资相关
- `薪资范围`：明确的薪资区间，如"15-25K/月"、"年薪30-40万"
- `最低薪资`：薪资下限，如"15K"
- `最高薪资`：薪资上限，如"25K"
- `薪资单位`：薪资单位，如"K/月"、"万/年"

### 2. 企业性质
- `企业性质`：企业类型，可选值：
  - 国企
  - 民企
  - 外企
  - 事业单位
  - 上市公司
  - 独角兽企业
  - 其他

### 3. 地理位置相关
- `工作地点`：城市或地区，如"北京"、"上海"、"深圳"
- `区域要求`：具体区域，如"朝阳区"、"浦东新区"

### 4. 学历要求
- `学历要求`：学历层次，可选值：
  - 博士
  - 硕士
  - 本科
  - 大专
  - 高中
  - 不限
- 也可使用范围表示：如"本科及以上"

### 5. 工作经验
- `工作经验`：工作年限，如"1-3年"、"3-5年"、"5年以上"
- 也可使用具体表述：如"应届生可"、"有相关经验"

### 6. 职位类型
- `职位类型`：岗位类别，如"技术岗位"、"产品岗位"、"运营岗位"、"市场岗位"

### 7. 时间相关
- `截止日期`：岗位申请截止日期，格式"YYYY-MM-DD"
- `发布时间范围`：岗位发布的时间范围，如"2024-01-01至2024-03-31"

### 8. 其他
- `公司规模`：公司人数规模，如"100-500人"、"500人以上"
- `行业领域`：所属行业，如"互联网"、"金融"、"教育"
- `关键词`：职位描述或公司名称中包含的关键词，如"Java"、"Python"

## 示例模板

### 模板1：基础筛选
```
薪资范围：15-25K/月
企业性质：国企
学历要求：本科及以上
工作经验：1-3年
```

### 模板2：地域+职位类型
```
工作地点：北京
职位类型：技术岗位
最低薪资：15K
学历要求：硕士
```

### 模板3：行业+经验
```
行业领域：互联网
公司规模：500人以上
工作经验：3-5年
关键词：后端开发
```

### 模板4：时间范围筛选
```
截止日期：2024-12-31
发布时间范围：2024-01-01至2024-06-30
工作地点：上海
薪资范围：20-40K/月
```

### 模板5：复杂条件组合
```
工作地点：北京或上海
企业性质：上市公司
职位类型：产品经理
最低薪资：20K
学历要求：本科及以上
工作经验：2-5年
公司规模：1000人以上
行业领域：互联网或金融
```

## 注意事项
1. **条件匹配规则**：所有条件为"与"关系，必须同时满足
2. **模糊条件处理**：
   - 薪资范围：如文档中写"15K+"，则视为满足最低薪资要求
   - 工作经验：如文档中写"1-3年或5年以上"，满足任一条件即视为符合
   - 学历：如文档中写"本科或硕士"，满足任一条件即视为符合
3. **关键词匹配**：关键词在职位名称、职位描述或公司名称中出现即视为匹配
4. **多值条件**：如"北京或上海"使用"或"连接，满足任一值即视为符合
5. **范围条件**：如"本科及以上"包含本科、硕士、博士等更高学历

# 主流开发框架速查

## 前端框架

### React
- **类型**: 前端 UI 库
- **适用场景**: 大型单页应用、需要高度组件化的项目
- **核心特性**: 虚拟 DOM、组件化、Hooks、生态系统丰富
- **学习曲线**: 中等

### Vue.js
- **类型**: 渐进式前端框架
- **适用场景**: 中小型项目、快速原型开发
- **核心特性**: 双向绑定、指令系统、单文件组件、易上手
- **学习曲线**: 平缓

### Svelte
- **类型**: 编译型前端框架
- **适用场景**: 性能敏感的应用、小型到中型项目
- **核心特性**: 编译时优化、无虚拟 DOM、响应式声明
- **学习曲线**: 平缓

## 后端框架

### Django (Python)
- **类型**: 全功能 Web 框架
- **适用场景**: 快速开发、内容管理系统、数据驱动应用
- **核心特性**: ORM、管理后台、认证系统、 batteries-included
- **学习曲线**: 中等

### FastAPI (Python)
- **类型**: 现代异步 Web 框架
- **适用场景**: API 服务、微服务、高性能应用
- **核心特性**: 异步支持、自动文档、类型提示、高性能
- **学习曲线**: 平缓

### Spring Boot (Java)
- **类型**: 企业级微服务框架
- **适用场景**: 大型企业应用、微服务架构
- **核心特性**: 自动配置、微服务支持、生态丰富、生产级
- **学习曲线**: 陡峭

### Gin (Go)
- **类型**: 高性能 Web 框架
- **适用场景**: 高性能 API、微服务、实时应用
- **核心特性**: 高性能、中间件支持、路由分组、错误处理
- **学习曲线**: 平缓

## 跨平台框架

### Flutter
- **类型**: UI 跨平台框架
- **适用场景**: 移动应用、桌面应用、嵌入式
- **核心特性**: 自绘引擎、热重载、丰富组件、Dart 语言
- **学习曲线**: 中等

### Electron
- **类型**: 桌面应用框架
- **适用场景**: 跨平台桌面应用
- **核心特性**: Web 技术栈、丰富 API、大型生态
- **学习曲线**: 平缓

## 选择建议

| 场景 | 推荐框架 |
|------|---------|
| 快速原型 | Vue.js, FastAPI |
| 大型企业应用 | Spring Boot, Django |
| 高性能 API | FastAPI, Gin |
| 跨平台移动 | Flutter |
| 跨平台桌面 | Electron |

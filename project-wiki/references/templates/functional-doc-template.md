# 功能文档模板

## 使用说明

这是一个功能文档模板，用于收集和整理用户的功能需求。

**填写指南**：
1. 使用此模板创建功能文档
2. 填写"用户填写部分"的内容
3. 智能体会自动生成"智能推断部分"
4. 保存到 `functional-docs/` 目录

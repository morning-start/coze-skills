## 数据流

```mermaid
sequenceDiagram
    participant A as 客户端
    participant B as API 层
    participant C as 业务层
    participant D as 数据层

    A->>B: 请求
    B->>C: 业务处理
    C->>D: 数据查询
    D-->>C: 返回数据
    C-->>B: 返回结果
    B-->>A: 响应
```

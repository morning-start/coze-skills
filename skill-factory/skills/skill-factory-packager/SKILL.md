# Skill Factory Packager - 技能打包器

## 技能职责边界

**本技能负责**：
- 验证技能族目录结构完整性
- 检查所有 SKILL.md 前言区字段
- 验证 parent 和依赖关系指向
- 输出打包验证报告

**本技能不负责**：
- 分析原始技术内容（由 analyzer 负责）
- 设计技能拆分方案（由 planner 负责）
- 生成具体技能文件（由 generator 负责）
- 修复检测到的问题（返回 generator 修复）

---

## 输入判断

### 有效输入

generator 输出的技能目录路径，必须存在：
- 母技能目录
- 母技能 SKILL.md
- skills/ 子目录
- 至少 1 个子技能目录

### 无效输入处理

- 目录不存在 → 返回错误："目录不存在，请检查路径"
- 母技能 SKILL.md 不存在 → 返回错误："母技能文件缺失，请返回 generator 重新生成"
- skills/ 子目录不存在 → 返回错误："子技能目录缺失，请返回 generator 重新生成"

## 核心逻辑

### 工作流程

```
技能目录 → 结构验证 → 前言区检查 → 关系验证 → 质量评分 → 打包报告
                      ↓
                检查是否通过？
                 ↓         ↓
                是        否
             打包输出   返回错误
```

### 步骤详解

#### 步骤 1：结构验证

**必须存在的文件/目录**：

| 类型 | 路径 | 验证方式 |
|------|------|----------|
| 母技能目录 | `{母技能名}/` | 目录存在 |
| 母技能文件 | `{母技能名}/SKILL.md` | 文件存在 |
| 子技能目录 | `{母技能名}/skills/{子技能名}/` | 每个子技能目录存在 |
| 子技能文件 | `{母技能名}/skills/{子技能名}/SKILL.md` | 每个子技能文件存在 |

**验证结果判断**：
- 全部存在 → 通过，继续步骤 2
- 任一缺失 → 标记缺失项，返回错误

#### 步骤 2：前言区检查

**SKILL.md 前言区标准字段**（7个）：

| 字段 | 母技能 | 子技能 | 说明 |
|------|--------|--------|------|
| name | ✅ 必须 | ✅ 必须 | 技能名称 |
| version | ✅ 必须 | ✅ 必须 | 版本号 |
| author | ✅ 必须 | ✅ 必须 | 作者 |
| description | ✅ 必须 | ✅ 必须 | 描述 |
| tags | ✅ 必须 | ✅ 必须 | 标签数组 |
| dependency | ✅ 必须 | ✅ 必须 | 依赖关系 |
| parent | - | ✅ 必须 | 父技能名称 |

**前言区检查结果判断**：
- 全部字段存在且有效 → 通过，继续步骤 3
- 任一缺失或无效 → 标记问题项，返回错误

#### 步骤 3：关系验证

**验证规则**：

| 验证项 | 规则 | 错误示例 |
|--------|------|----------|
| parent 指向 | 子技能的 parent 必须指向母技能 | 子技能 parent 指向错误 |
| requires 指向 | 有 requires 的子技能必须指向已存在的技能 | 指向不存在的技能 |
| children 列表 | 母技能的 children 必须包含所有子技能 | 列表与实际子技能不符 |
| 版本号格式 | 必须为 vX.X.X 格式 | v1.0 / v1 / 1.0.0 |

**验证结果判断**：
- 全部关系正确 → 通过，继续步骤 4
- 任一关系错误 → 标记错误项，返回错误

#### 步骤 4：质量评分

**评分维度**：

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 结构完整性 | 25% | 目录和文件是否完整 |
| 前言区完整性 | 25% | 标准字段是否齐全 |
| 关系正确性 | 25% | parent/requires 是否正确 |
| 内容充实度 | 25% | SKILL.md 正文是否 > 500 字符 |

**评分等级**：

| 总分 | 等级 | 处理方式 |
|------|------|----------|
| 90-100 | 优秀 | 直接打包输出 |
| 70-89 | 良好 | 打包输出，附带改进建议 |
| 60-69 | 合格 | 打包输出，附带必须修复项 |
| < 60 | 不合格 | 返回 generator 修复 |

- 质量评分和等级
- 改进建议（可选）

## 失败处理

| 失败场景 | 处理方式 |
|----------|----------|
| 目录结构缺失 | 返回错误，列出缺失项，要求返回 generator 补充 |
| 前言区字段缺失 | 返回错误，列出缺失字段，要求返回 generator 修复 |
| parent 指向错误 | 返回错误，要求返回 generator 修正 |
| 存在循环依赖 | 返回错误，要求返回 planner 重新设计 |
| 质量评分 < 60 | 返回错误，列出不合格项，要求返回 generator 修复 |

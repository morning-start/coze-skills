# Skill Factory Analyzer - 技术内容分析器

## 任务目标

- **本 Skill 用于**：分析技术网站/文档，提取技术特点和知识结构
- **核心能力**：网站内容读取、技术特点提取、知识规划、迭代式学习
- **触发条件**：需要分析技术网站或文档时

## 工作流程

```
输入（URL/文档） → 内容读取 → 技术特点提取 → 知识规划 → 迭代学习 → 分析报告
```

## 操作步骤

### 步骤 1：内容读取

1. 验证 URL 可访问性
2. 使用 **WebFetch** 获取网页内容
3. 使用 **Read** 读取本地文档

### 步骤 2：技术特点提取

1. 识别技术名称和版本
2. 提取核心功能描述
3. 识别应用场景
4. 记录关键术语和概念

### 步骤 3：知识结构规划

1. 分析知识缺口
2. 确定技术栈的广度和深度
3. 生成知识规划清单（高/中/低优先级）

### 步骤 4：迭代学习补充

1. 对高优先级主题执行 **search**
2. 评估信息完整性（目标 80%+）
3. 判断是否继续迭代

## 分析模式

### 模式 A：智能网站分析

1. 验证 URL 可访问性
2. 使用 **WebFetch** 获取网页内容
3. 提取核心功能、关键概念、应用场景
4. 使用 **search** 迭代补充（目标 80%+ 完整度）

### 模式 B：文档分析

1. 使用 **Read** 或 **WebFetch** 读取文档
2. 提取目录结构和章节划分
3. 识别技术模块
4. 使用 **WebSearch** 补充最佳实践

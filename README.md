# Coze Skills 技能集合

这是一个为 Coze AI 助手设计的技能集合，包含多个专业领域的实用技能，帮助提升开发效率和代码质量。

## 技能列表

### 1. API 文档自动生成器 (api-doc-generator)

自动分析项目代码并生成符合标准格式的API文档。

**核心功能：**
- 支持 Flask/FastAPI/Express 等主流框架
- 自动划分功能模块和层级结构
- 生成包含概述、认证、端点详情、错误处理等完整章节的文档
- 代码扫描、路由识别、模块划分、文档结构化

**适用场景：**
- 需要为现有项目生成 API 文档
- 需要快速了解项目的 API 结构
- 需要维护和更新 API 文档

**技术栈：** Python (ast, re, json, pathlib)

---

### 2. 软著申请辅助技能 (copyright-assist)

辅助软件著作权申请，包括设计说明书、用户说明书、操作手册撰写指导及源代码提取格式化。

**核心功能：**
- 源代码提取与格式化（符合软著要求的60页代码）
- 设计说明书/用户说明书/操作手册撰写指导
- 资源完整性检查与缺失提醒
- 说明书模板填充与格式调整
- 支持多版本申请、智能资源检查、自动打包符合官方要求的材料

**适用场景：**
- 需要申请软件著作权
- 已有项目代码、文档或部分材料
- 需要指导如何准备软著申请材料

**技术栈：** Python (chardet, PyYAML)

---

### 3. 项目知识库构建工具 (project-wiki)

智能项目知识库构建工具，自动分析代码提取隐性知识，生成结构化文档与架构图谱。

**核心功能：**
- 架构解析：自动识别项目结构、模块划分、技术栈
- 隐性知识挖掘：提取代码中的设计模式、架构决策、最佳实践
- 知识图谱构建：生成模块/API/概念的关系网络（Mermaid 可视化）
- 结构化文档生成：README、API 文档、架构文档自动生成

**适用场景：**
- 新人入职：快速理解项目架构和核心模块
- 代码重构：评估影响范围，识别技术债
- 知识传承：将资深开发者的隐性知识显性化
- 架构评审：可视化展示系统设计

**支持的框架：** 13+ 主流框架（React/Vue/Django/Spring Boot/Flask/FastAPI/Gin/Flutter/Electron/Tauri/Wails/Svelte/SolidJS）

---

### 4. Python 团队协同开发技能 (python-team)

通过自主学习、PM、架构师、高级程序员四角色协同，从自然语言需求自动生成完整Python项目交付物。

**核心功能：**
- 自主学习与知识整合、需求分析与澄清
- 架构设计（自适应项目规模）
- 代码实现、功能验证、版本控制
- 功能扩展、项目重构、技能识别与调用
- 数据库设计与实现（SQLite、PostgreSQL、MongoDB、向量数据库、图数据库）
- 数据层抽象（Repository 模式）、数据库切换

**适用场景：**
- 软件需求澄清、快速原型开发
- 项目初始化、功能扩展、代码重构

**技术栈：** Python 3.11+、UV 包管理、loguru 日志、SQLAlchemy、MongoDB

---

### 5. Pythonic 代码风格 (pythonic-style)

Python代码风格优化与Pythonic惯用法指导。

**核心功能：**
- 代码风格分析：识别非 Pythonic 模式，提供改进建议
- Pythonic 惯用法：列表推导、生成器、上下文管理器、装饰器、元类等
- 设计模式应用：SOLID 原则、描述符协议、迭代器协议等
- 性能优化：内存优化、计算优化、I/O 优化、并发模式、缓存策略
- 重构指导：代码异味检测、重构技巧、重构模式、重构流程
- 实战模板：提供可直接使用的代码模板（140+ 个）

**核心理念：** Friendly Python = User-Friendly + Maintainer-Friendly

**适用场景：**
- 代码审查和风格改进
- Python 特定语法的最佳实践
- 性能优化或重构建议
- 高级 Python 模式或设计模式

### 6. 招聘信息处理技能 (recruitment-processor)

自动处理多份招聘markdown文档，识别文档中的图片内容，提取关键信息，根据用户条件筛选，生成结构化总结报告。

**核心功能：**
- 图片识别与OCR、文本理解、信息提取
- 条件匹配、报告生成
- 批量处理招聘文档
- 提取职位、薪资、截止时间等关键信息

**适用场景：**
- 批量处理招聘信息
- 根据条件筛选职位
- 生成结构化的招聘总结报告

## 快速开始

### 使用技能

每个技能都有独立的 `SKILL.md` 文件，包含详细的使用说明、操作步骤和示例。

1. 进入对应的技能目录
2. 阅读 `SKILL.md` 文件
3. 按照说明使用技能

### 技能结构

每个技能目录通常包含以下结构：

```
skill-name/
├── SKILL.md              # 技能主文档（必读）
├── assets/               # 模板和资源文件
│   └── templates/        # 代码模板
├── references/           # 参考文档
│   └── *.md             # 各类参考指南
├── scripts/              # Python 脚本（如有）
│   └── *.py             # 辅助脚本
└── config.yaml           # 配置文件（如有）
```

## CI/CD 自动化

项目使用 GitHub Actions 实现自动化构建和发布流程。

### 快速发布

```bash
# 1. 更新版本号
uv run bump-version 1.1.0 --commit --tag

# 2. 推送到远程仓库
git push origin master
git push origin v1.1.0
```

推送 tag 后，GitHub Actions 会自动：
- 构建所有技能为 `.skill` 格式
- 生成发布说明
- 创建 GitHub Release
- 上传构建产物

### 本地构建

```bash
# 构建所有技能
uv run build-skills --version 1.0.0

# 构建指定技能
uv run build-skills --version 1.0.0 --skill recruitment-processor
```

### 详细文档

查看 [CI/CD 使用指南](docs/CI_CD_GUIDE.md) 了解更多详情。
